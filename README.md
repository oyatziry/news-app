# Newsify

Newsify is a news search app where the user can find out about the latest news just by landing on the homepage, searching for a topic, or sorting by categories. 

Once the user has created an account they will have the ability to add articles they are interested in or would like to read later to their bookmark. The user may add as many or even delete articles from their bookmarks. 

Newsify is the number 1 news search app in GA for Cohort SEIR-119! If the user is not satisfied they can always delete their profile or just edit their name or email. So, get your news now!

# Features

* As a user I want to enter a login page 
* As a user I want to enter a homepage after logging in with the latest news
* As a user I want to be able to click on a article to be taken to the external page of the article
* As a user I would like a search bar to search for articles by topic, name, or keyword
* As a user I would like to sort by category
* As a user I would like to add articles to a bookmark
* As a user I would to delete articles from bookmark page
* As a user I would like to access homepage first then be given the option to login later
* As a user I would like to be able to edit the name and email of profile
* As a user I would like to be able to delete profile
